# 📈 Trade Performance vs Market Sentiment Analysis

> A data analysis project that explores the relationship between trade performance and market sentiment using Python, exploratory data analysis (EDA), and classification techniques.

---

## 📌 Project Overview

Understanding how market sentiment influences trade performance is essential for making informed financial decisions. This project analyzes trading data to uncover patterns, trends, and relationships between market sentiment and trading outcomes.

The project includes:

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Feature engineering
- Classification using Random Forest
- Performance evaluation

---

## 🎯 Objectives

- Analyze the relationship between trade performance and market sentiment.
- Explore trends through visualizations.
- Prepare the dataset for predictive analysis.
- Build a classification model to evaluate trade performance.
- Interpret model results and derive meaningful insights.

---

The dataset contains information related to:
